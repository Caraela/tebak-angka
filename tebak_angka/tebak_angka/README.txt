PERMAINAN TEBAK ANGKA (SHELL)

Cara Menjalankan:
1. Buka terminal
2. Masuk ke folder tempat file berada
3. Beri izin eksekusi:
   chmod +x tebak_angka.sh
4. Jalankan:
   ./tebak_angka.sh

Fitur:
- Level kesulitan (Mudah, Sedang, Sulit)
- Mode 1 pemain & 2 pemain
- Skor disimpan otomatis ke skor.txt
- Bisa reset skor
